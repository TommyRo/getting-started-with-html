HTML (Hypertext Markup Language)
The basic structure of of an HTML tag:
<tag name>contents</tag name>

Here are some tag examples:
- for BOLD: b. Example: <b>Hello</b>. This will make the word "Hello" bold.
- for ITALICS: em
- for insertion of hyperlink: <a href = "www.cnn.com">News</a>
- If you click on "News" the website www.cnn.com will open.
- for IMAGES: <img scr="url">. The "url" here will contain the path to the image. No closing tag needed.
- Line break: br. No closing tag needed.
- Paragraph: <p>contents</p>

There is a difference between inline and block-level tags.
